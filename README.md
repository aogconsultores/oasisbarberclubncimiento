# Oasis Barber Club — Sitio Web

Sitio web oficial de **Oasis Barber Club**, Nacimiento, Bío Bío, Chile.

## Estructura

```
oasis-web/
├── index.html              ← Página principal
├── README.md
└── assets/
    └── images/
        ├── hero.jpg        ← Foto hero (interior barbería)
        ├── darwin.png      ← Foto Darwin (barbero)
        ├── joherman.jpg    ← Foto Joherman (barbero)
        ├── filosofia-main.jpg  ← Foto sección Nuestra Filosofía
        ├── logo-negro.png  ← Logo Oasis fondo negro
        ├── silla.jpg       ← Foto silla premium
        ├── corte-barba.jpg ← Galería: corte con barba
        ├── corte-navaja.png← Galería: navaja
        ├── corte-nino.jpg  ← Galería: diseño niño
        ├── color1.jpg      ← Galería: color
        └── color2.jpg      ← Galería: color artístico
```

## Deploy en GitHub Pages

1. Sube todos los archivos a un repositorio público
2. Ve a **Settings → Pages**
3. Source: `Deploy from branch` → `main` → `/ (root)`
4. La web quedará en: `https://TU_USUARIO.github.io/NOMBRE_REPO`
